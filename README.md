# Weather App

A responsive React app using OpenWeather API to search for city weather and view current details + 5-day forecast.

## Setup
1. Replace API_KEY in `src/utils/api.js`.
2. `npm install`
3. `npm start`

## Features
- Search by city
- Current weather (temp, feels like, humidity, description)
- 5-day forecast (approx daily summaries)
- Responsive design for mobile/desktop
